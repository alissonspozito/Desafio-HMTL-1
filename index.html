Estrutura Básica do HTML

(voltar)

De início, foi criado um novo arquivo em index.html. O HTML consiste em três partes principais: o próprio HTML, head e body. O que estiver entre o campo body é o que aparece ao usuário, enquanto o head contém elementos que pré-carregam antes para o usuário.

"As tags são elementos fundamentais usados para estruturar e definir o conteúdo de uma página web. Elas são delimitadas por colchetes angulares (< >) e geralmente vêm em pares, como (abertura) e (fechamento), com o conteúdo entre elas. Algumas tags, como < img > ou < br >, são autossuficientes e não precisam de fechamento."

Os atributos do HTML são usados para adicionar informações adicionais às tags, permitindo configurar e personalizar seu comportamento e aparência. Eles definem propriedades como identificação, classes, estilos, funcionalidade ou dados específicos, adaptando os elementos às necessidades do conteúdo e da interação com os usuários.
